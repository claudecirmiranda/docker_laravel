Introdução

URL Base: http://localhost

Esta documentação provê informações sobre o uso do micro serviço de rastreamento de pedidos da Nagem.

Conforme você rola a página, verá exemplos de código para trabalhar com a API em diferentes linguagens de programação na área escura à direita (ou como parte do conteúdo no mobile). Você pode alternar a linguagem utilizada através das abas no canto superior direito (ou pelo menu de navegação no canto superior esquerdo no mobile).

Autenticando requisições

Para autenticar as requisições, inclua um header Authorization com o valor "Bearer {YOUR_AUTH_KEY}".

Todos os endpoints que requerem autenticação são marcados com o selo requer autenticação na documentação abaixo.

Você pode gerar o token atráves do sistema de autenticação interno da Nagem.
